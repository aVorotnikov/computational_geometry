# Триангуляция многоугольника
Осуществляется триангуляции "правильного" многоугольника за квадратичное время.

## Вход
Вход состоит из `n` строк которые задают `n`-угольник. Каждая строка содержит две целочисленные координаты.

**Пример.** Единичный четырёхугольник:
```
0 0
1 0
1 1
0 1
```

## Выход
Выход содержит `n-2` строки, каждая задаёт треугольник. Каждая строка содержит три индекса вершины, которые определяются
согласно входу: вершина, соответствующая первой строки имеет индекс `0`, последняя - `n-1`.

**Пример.**
```
0 1 2
2 3 0
```
